# 如果您收到通知
### 如果有人追蹤您，請尋求專業協助

如果 AirGuard 在背景偵測到一個跟隨您的追蹤器，您將收到通知。您可以點擊通知以瞭解有關此追蹤器的更多資訊。對於支援的追蹤器，您可以播放聲音以定位追蹤器。對於所有其他追蹤器，您可以使用「定位追蹤器」來找到它。

如果您找到了追蹤器，並且確定它具有惡意，請嘗試取出電池。並確保在停用它之前不要移動到其他位置。

如果您不幸發現了一個針對您的追蹤器，重要的是要知道在哪裡獲得幫助。
有幾個網絡和協會可以幫助您。我們按國家/地區提供一個簡短的列表。
