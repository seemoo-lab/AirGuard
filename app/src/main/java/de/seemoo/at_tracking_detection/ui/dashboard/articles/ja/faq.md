# FAQ

## 一般とコア機能

**どの種類のトラッカーを検出できますか？**
現在、Apple AirTags、AirPods、FindMyデバイス、紛失モードの他のAppleデバイス、Samsung SmartTags、Chipolos、Tilesなど、さまざまなトラッカーを検出しています。

**AirGuardはどれくらいバッテリーを消費しますか？**
AirGuardは電力効率が良いように設計されています。通常の使用では、バッテリー寿命に大きな違いは見られません。通常、バッテリーの1％未満しか使用しません。

**AirGuardは無料のままですか？**
はい、AirGuardを無期限に無料に保つ予定です。

## 検出について

**近くにトラッカーが表示された場合、どうすればよいですか？**
手動検索であなたの地域にトラッカーが表示されても、すぐに心配する必要はありません。これらのトラッカーは単に近くにあるだけで、必ずしもあなたを追跡しているわけではありません。これらのトラッカーのいずれかがあなたと一緒に移動している場合、私たちのアプリAirGuardが警告します。そうでない場合は、通常、心配する必要はありません。

**白と灰色のデバイスの違いは何ですか？**
白いデバイス（またはダークモードでは黒）は、所有者から切断された「オフライン」モードです。これらは追跡の脅威になる可能性があります。私たちはそれらの位置を保存し、あなたを追跡しているように見える場合は警告します。灰色のデバイスは、所有者のデバイスに接続されているか、最近接続されていたものです。これらは一般的に安全であり、アプリには保存されません。

**「接続されたデバイス」とはどういう意味ですか？**
接続されたデバイスとは、現在または最近（15分以内）に所有者のデバイスに接続されたデバイスのことです。これらは所有者の近くにあるためリスクはなく、アプリには保存されません。

**所有者に接続されているデバイスは安全ですか？**
はい、所有者にリンクされているデバイスは、独立して位置データを送信できないため安全です。これにより、不要な追跡やストーカー行為には不向きです。

**接続されたデバイスのデータを保存しないのはなぜですか？**
接続されたデバイスは通常、所有者と一緒に移動しているため、追跡の脅威にはなりません。たとえば、電車の中の誰かがバッグにiPhoneとAirTagを入れている場合、アラートは表示されません。AirTagがあなたを追跡していないと想定しています。しかし、近くにiPhoneがない場合、AirTagは懸念事項になる可能性があります。

**一部のトラッカーが2回リスト表示されるのはなぜですか？**
トラッカーは定期的にBluetooth IDを変更することがあるため、2回表示されることがあります。これにより、一貫して追跡することが難しくなり、重複したリストが表示されます。

**自分のAirTagがアプリに常に表示されないのはなぜですか？**
時々、あなたのAirTagがアプリに表示されないことがあります。ただし、他の誰かのApple IDにリンクされていて、あなたのデバイスに接続されていない場合は、常に表示されます。

## 追跡アラートとリスク

**アプリはどのように追跡リスクを評価しますか？**
アプリは追跡リスクレベル（リスクなし、中程度のリスク、高いリスク）を計算します。これは、追跡デバイスがどれくらいの頻度で、どこであなたの近くにあったかなど、いくつかの要因に基づいています。高いリスクが必ずしも追跡されていることを意味するわけではないので、常にトラッカーのマップを確認してください。

**追跡アラートを受け取った場合、どのような手順を踏むべきですか？**
1. トラッカーを手動で検索します。
2. 可能であれば、音を鳴らして場所を特定します。
3. アプリのマップを調べて、どれくらいの期間あなたを追跡しているかを確認します。
4. トラッカーを無効にするか、可能であればバッテリーを取り外します。取り外しができない場合（一部のTileトラッカーなど）、アルミホイルで包んで信号を弱めます。
5. トラッカーを持って警察に報告します。
6. トラッカーが無効になるまで、自宅のような安全な場所に行かないでください。

**アラートの後、トラッカーが見つからない場合はどうすればよいですか？**
特に公共交通機関などの混雑した場所では、誤報が発生することがあります。異なる日に繰り返しアラートが表示される場合は、再確認してください。音機能、信号強度インジケーター、または詳細スキャンを利用してデバイスを見つけてください。

**トラッカーの音を鳴らすのに問題があるのはなぜですか？**
この問題は、トラッカーから遠すぎる場合に発生する可能性があります。近づいて、信号強度や詳細スキャンを使用して場所を特定してください。この問題は、検索中にトラッカーが範囲外に移動したり、所有者に再接続したりした場合にも発生する可能性があります。また、トラッカーが内部状態に応じて音を鳴らさないことを決定する場合もあります。

## アプリ設定とデバイス管理

**設定の「検出感度」は何をしますか？**
この設定は、トラッカーがリスクと見なされるまでにどれくらいの期間あなたを追跡する必要があるかを調整します。感度が低いほど、より長く追跡する必要があり、感度が高いほど、デバイスをより迅速にフラグ立てします。

**低電力スキャンモードを有効にするとどのような影響がありますか？**
このモードはスキャン頻度を減らし、エネルギーを節約しますが、検出効果は低下します。一部の危険なデバイスを見逃す可能性があります。安全性を高めるために通常モードを使用することをお勧めします。これは、最新のスマートフォンではバッテリーを大幅に消耗することはありません。

**AirGuardの位置情報アクセスを制限するとどうなりますか？**
位置情報アクセスがないと、アプリはBluetoothスキャンのみに依存します。これにより、トラッカーに対するあなたの動きを追跡できないため、誤報が増える可能性があります。より正確なアラートのために、位置情報アクセスを許可することをお勧めします。

**自分のトラッカーを無視したり、ラベルを付けたりできますか？**
現在、技術的な制限により、Tileブランドのトラッカーのみを無視できます。トラッカーにラベルを付けることはできますが、同じトラッカーがIDを変更して新しいトラッカーとして認識される可能性があることに注意してください。後で識別するためだけに名前を付けることをお勧めします。

**一部のデバイスを無視できないのはなぜですか？**
識別子を頻繁に変更するデバイスは無視できません。キーを変更すると、システムはそれらを新しいデバイスとして扱い、新しいアラートをトリガーします。一部のデバイス（Samsung SmartTagsなど）はキーを頻繁に変更しますが、変更しないデバイス（ChipolosやTilesなど）は無視できます。
