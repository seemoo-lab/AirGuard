# Jak AirGuard funguje?

Chytré sledovače předmětů používají Bluetooth k připojení k cizím zařízením. Tato zařízení pak odesílají svou polohu na internet. Díky tomu může majitel chytrého sledovače předmětů vidět polohu sledovače, i když je míle daleko.

Proto mohou být tyto chytré sledovače předmětů zneužity ke sledování lidí. Aby vás AirGuard ochránil před takovým útokem, používá Bluetooth ke skenování okolních sledovačů. Aplikace využívá vaši polohu k detekci, zda se s vámi sledovač pohybuje.

Pokud AirGuard detekuje podezřelý sledovač, obdržíte oznámení s podrobnějšími informacemi.
