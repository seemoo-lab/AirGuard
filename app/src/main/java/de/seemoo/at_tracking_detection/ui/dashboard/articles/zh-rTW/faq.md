# 常見問題 (FAQ)

## 一般與核心功能

**你們能偵測哪些類型的追蹤器？**
我們目前偵測各種追蹤器，包括 Google「尋找我的裝置」裝置、Apple AirTag、AirPods、「尋找」裝置、處於遺失模式的其他 Apple 裝置、Samsung SmartTags、Chipolo 和 Tile。

**AirGuard 耗電量如何？**
AirGuard 旨在節能。在一般使用情況下，您不會注意到電池壽命有顯著差異；它通常使用少於 1% 的電池。

**AirGuard 會永遠免費嗎？**
是的，我們計劃讓 AirGuard 無限期免費。

## 了解偵測結果

**如果我在附近看到追蹤器該怎麼辦？**
如果您透過手動搜尋注意到所在區域有追蹤器，請不要立即擔心。這些追蹤器只是在附近，不一定在跟蹤您。我們的應用程式 AirGuard 會在這些追蹤器中的任何一個與您一起移動時向您發出警報。如果沒有，通常沒什麼好擔心的。

**白色和灰色裝置有什麼區別？**
白色裝置（或深色模式下的黑色）處於「離線」模式，與其擁有者斷開連線。這些可能是追蹤威脅。我們會儲存它們的位置，並在它們似乎在跟蹤您時向您發出警報。灰色裝置是已連線或最近已連線到擁有者裝置的裝置。它們通常是安全的，不會儲存在應用程式中。

**「已連線裝置」是什麼意思？**
已連線裝置是指目前或最近（15 分鐘內）已連線到其擁有者裝置的裝置。這些不會儲存在應用程式中，因為它們在擁有者附近，沒有風險。

**與擁有者連線的裝置安全嗎？**
是的，與擁有者連結的裝置是安全的，因為它們無法獨立發送位置資料。這使得它們不適合用於不必要的追蹤或跟蹤騷擾。

**為什麼你們不儲存已連線裝置的資料？**
已連線裝置通常隨擁有者一起移動，因此不是追蹤威脅。例如，如果火車上有人在包裡放了 iPhone 和 AirTag，您不會收到警報。我們假設 AirTag 沒有在追蹤您。但是，如果附近沒有 iPhone，AirTag 可能是個隱憂。

**為什麼有些追蹤器會列出兩次？**
追蹤器可能會出現兩次，因為它們可能會定期更改其藍牙身分。這使得難以一致地追蹤它們，從而導致重複列表。

**為什麼我不總是在應用程式中看到我的 AirTag？**
有時，您的 AirTag 可能不會顯示在應用程式中。但是，如果它們連結到其他人的 Apple ID 且未連線到您的裝置，則始終可見。

## 追蹤警報與風險

**應用程式如何評估追蹤風險？**
應用程式計算追蹤風險等級：無風險、中等風險和高風險。這是基於多個因素，包括追蹤裝置在您附近的頻率和位置。高風險並不總是意味著您被追蹤，因此請務必檢查追蹤器的地圖。

**如果收到追蹤警報，我該採取什麼步驟？**
1. 手動搜尋追蹤器。
2. 如果可能，啟動其聲音以定位它。
3. 檢查應用程式的地圖，查看它跟隨您多久了。
4. 停用追蹤器或取出其電池。如果不可拆卸（如 Tile 追蹤器），請用鋁箔紙將其包裹以減弱訊號。
5. 帶著追蹤器向警方報案。
6. 在追蹤器停用之前，不要去安全的地方，如您的家。

**如果我在警報後找不到追蹤器怎麼辦？**
可能會發生誤報，尤其是在公共交通等擁擠區域。如果您在不同日期收到重複警報，請再次檢查。利用聲音功能、訊號強度指示器或詳細掃描來定位裝置。

**為什麼我可能無法啟動追蹤器的聲音？**
如果您距離追蹤器太遠，可能會出現此問題。靠近一點，並使用訊號強度或詳細掃描來幫助您定位它。如果追蹤器在您搜尋期間移出範圍或連線到其擁有者，也可能會出現問題。有時，追蹤器也可能根據其內部狀態決定不播放聲音。

## 應用程式設定與裝置管理

**設定中的「偵測靈敏度」有什麼作用？**
此設定調整追蹤器必須跟隨您多久才會被視為風險。較低的靈敏度意味著它必須跟隨您更長時間；較高的靈敏度會更快標記裝置。

**啟用低功耗掃描模式有什麼影響？**
此模式降低掃描頻率，節省能源但降低偵測效果。它可能會遺漏一些有風險的裝置。我們建議使用正常模式以獲得更好的安全性，這在現代手機上不會顯著消耗更多電池。

**如果我限制應用程式的位置存取權限會怎樣？**
沒有位置存取權限，應用程式僅依靠藍牙掃描。這可能會導致更多誤報，因為我們無法追蹤您相對於追蹤器的移動。我們建議允許位置存取權限以獲得更準確的警報。

**我可以忽略或標記我自己的追蹤器嗎？**
目前，由於技術限制，您只能忽略 Tile 品牌的追蹤器。您可以標記追蹤器，但請注意，同一個追蹤器可能會更改其 ID，並被視為新追蹤器。我們建議僅為了日後識別而命名它們。

**為什麼我不能忽略某些裝置？**
經常更改其識別碼的裝置無法被忽略。當它們更改金鑰時，我們的系統會將其視為新裝置，觸發新警報。某些裝置（如 Samsung SmartTags）經常更改金鑰，但您可以忽略不更改金鑰的裝置（如 Chipolo 和 Tile）。
