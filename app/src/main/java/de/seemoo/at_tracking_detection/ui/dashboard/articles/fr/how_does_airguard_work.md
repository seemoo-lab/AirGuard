# Comment fonctionne AirGuard ?

Les trackers d'objets intelligents utilisent le Bluetooth pour se connecter à des appareils étrangers. Ces appareils envoient ensuite leur localisation à Internet. Par conséquent, le propriétaire du tracker d'objets intelligent peut alors voir la localisation du tracker même s'il se trouve à des kilomètres.

C'est pourquoi ces trackers d'objets intelligents peuvent être détournés pour harceler les gens. Pour vous protéger contre une telle attaque, AirGuard utilise le Bluetooth pour rechercher les trackers à proximité. L'application utilise votre localisation pour détecter si un tracker se déplace avec vous.

Si AirGuard détecte un tracker suspect, vous recevrez une notification avec des informations plus détaillées.
