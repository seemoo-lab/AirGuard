# AirGuardはどのように機能しますか？

スマートアイテムトラッカーは、Bluetoothを使用して他のデバイスに接続します。これらのデバイスは、その位置情報をインターネットに送信します。その結果、スマートアイテムトラッカーの所有者は、トラッカーが何マイルも離れていてもその場所を見ることができます。

そのため、これらのスマートアイテムトラッカーは、人々をストーカーするために悪用される可能性があります。このような攻撃からあなたを守るために、AirGuardはBluetoothを使用して近くのトラッカーをスキャンします。アプリはあなたの位置情報を使用して、トラッカーがあなたと一緒に移動しているかどうかを検出します。

AirGuardが不審なトラッカーを検出すると、詳細情報が記載された通知が届きます。
