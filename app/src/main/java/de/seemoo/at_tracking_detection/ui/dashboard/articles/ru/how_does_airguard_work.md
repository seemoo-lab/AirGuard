# Как работает AirGuard?

Умные трекеры вещей используют Bluetooth для подключения к чужим устройствам. Эти устройства затем отправляют свое местоположение в Интернет. В результате владелец умного трекера может видеть местоположение трекера, даже если он находится за многие километры.

Вот почему эти умные трекеры могут быть использованы для преследования людей. Чтобы защитить вас от такой атаки, AirGuard использует Bluetooth для сканирования находящихся поблизости трекеров. Приложение использует ваше местоположение, чтобы определить, движется ли трекер вместе с вами.

Если AirGuard обнаружит подозрительный трекер, вы получите уведомление с более подробной информацией.
