# このアプリの制限事項

AirGuardはBluetoothベースのトラッカーしか見つけることができません。GPSで動作し、携帯電話接続を介して所有者と位置情報を共有するトラッカーは見つけることができません。

以下では、AirGuardがバックグラウンドで自動的に検出できるトラッカーと、手動スキャンで表示されるトラッカーについて説明します。

**Samsung SmartTags**
これらのタグはSamsung製で、AirTagとほぼ同じように機能します。

*自動追跡アラート* ✅
*手動スキャン* ✅

**Tile トラッカー**
TileはBluetoothトラッカーを専門に製造している会社です。これらもAirTagと同様の機能を備えています。

*自動追跡アラート* ✅
*手動スキャン* ✅

**Chipolo トラッカー（Find My非対応）**
Chipoloは、iPhoneでのみ動作するFind Myトラッカーと、Androidデバイスでも動作するトラッカーを提供しています。ここでは、Androidスマートフォンとも互換性のあるトラッカーについて説明します。

*自動追跡アラート* ✅
*手動スキャン* ✅

**AirTags**
AirTagはAppleのトラッカーです。その結果、多くの信号がシステムによってフィルタリングされるため、検出は限定的です。手動スキャンではAirTagを見つけることができ、位置を特定できます。バックグラウンドでの追跡検出は、すでにAppleの「探す」（Find My）アプリによって提供されています。

*自動追跡アラート* ❌
*手動スキャン* ✅

**Find My トラッカー**
これらは、Appleの「探す」（Find My）と互換性がありますが、他のメーカーから提供されているトラッカーです。Chipolo、Pebblebee、Atuvosなど、このようなトラッカーを製造しているメーカーがいくつかあります。AirTagと同様に、トラッカーをバックグラウンドで検出することはできません。この場合も、「探す」（Find My）アプリが自動的に警告します。

*自動追跡アラート* ❌
*手動スキャン* ✅

現在、すべてのトラッカータイプで、所有者に接続されているかどうかを判断できるわけではないことに注意してください。そのため、自分自身や友人のトラッカーが表示される場合があります。
また、一部のトラッカーは1日に数回、ID（Bluetooth MACアドレス）をランダムに変更することが重要です。その結果、1つのトラッカーがアプリに複数回表示される可能性があります。これは、異なるランダム化されたIDを照合することが多くの場合不可能だからです。
