# Ako funguje AirGuard?

Inteligentné sledovače predmetov používajú Bluetooth na pripojenie k cudzím zariadeniam. Tieto zariadenia potom odosielajú svoju polohu na internet. Vďaka tomu môže majiteľ inteligentného sledovača predmetov vidieť polohu sledovača, aj keď je vzdialený míle ďaleko.

Preto môžu byť tieto inteligentné sledovače predmetov zneužité na sledovanie ľudí. Aby vás AirGuard ochránil pred takýmto útokom, používa Bluetooth na skenovanie okolitých sledovačov. Aplikácia využíva vašu polohu na zistenie, či sa s vami sledovač pohybuje.

Ak AirGuard zistí podozrivý sledovač, dostanete upozornenie s podrobnejšími informáciami.
