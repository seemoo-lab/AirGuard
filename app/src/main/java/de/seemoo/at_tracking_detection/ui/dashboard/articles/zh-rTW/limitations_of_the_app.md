# 本應用程式的限制

AirGuard 只能找到基於藍牙的追蹤器。無法找到使用 GPS 並透過行動網路與擁有者分享位置的追蹤器。

接下來，您將瞭解 AirGuard 可以在背景自動偵測哪些追蹤器，以及哪些追蹤器可透過手動掃描看到。

**Samsung SmartTags**
這些標籤由三星製造，運作方式與 AirTags 幾乎相同。

*自動追蹤警報* ✅
*手動掃描* ✅

**Tile 追蹤器**
Tile 是一家專門製造藍牙追蹤器的公司。這些追蹤器也具有與 AirTags 類似的功能。

*自動追蹤警報* ✅
*手動掃描* ✅

**Chipolo 追蹤器 (不含 Find My)**
Chipolo 提供僅適用於 iPhone 的 Find My 追蹤器，以及也適用於 Android 裝置的追蹤器。這裡我們涵蓋的是也與 Android 手機相容的追蹤器。

*自動追蹤警報* ✅
*手動掃描* ✅

**AirTags**
AirTags 是 Apple 的追蹤器。因此，它們只能在有限的程度上被偵測到，因為許多訊號被系統過濾了。在手動掃描中，我們可以找到 AirTags 並且您可以定位它們。背景追蹤偵測已由 Apple 的「尋找」(Find My) 應用程式提供。

*自動追蹤警報* ❌
*手動掃描* ✅

**Find My 追蹤器**
這些是與 Apple 的「尋找」網路相容，但來自其他製造商的追蹤器。有幾家製造商生產此類追蹤器：Chipolo、Pebblebee、Atuvos 等。與 AirTags 一樣，無法在背景偵測到該追蹤器。同樣，「尋找」應用程式會自動警告您。

*自動追蹤警報* ❌
*手動掃描* ✅

請注意，目前無法確定所有類型的追蹤器是否已連線到其擁有者。因此，您可能會看到自己或朋友的追蹤器。
此外，很務必知道的是，某些追蹤器每天會隨機更改其身份（藍牙 MAC 位址）數次。因此，一個追蹤器可能會在應用程式中多次顯示。這是因為通常無法匹配不同的、隨機化的身份。
