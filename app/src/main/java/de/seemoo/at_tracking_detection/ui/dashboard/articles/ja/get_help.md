# 通知を受け取った場合
### 誰かに追跡されている場合は、専門家の助けを求めてください

AirGuardがバックグラウンドであなたを追跡しているトラッカーを検出すると、通知が届きます。通知をタップすると、そのトラッカーに関する詳細情報を確認できます。サポートされているトラッカーの場合は、音を鳴らしてトラッカーを見つけることができます。他のすべてのトラッカーについては、「トラッカーを特定」を使用して見つけることができます。

トラッカーを見つけ、それが悪意のあるものであると確信した場合は、バッテリーを取り外してみてください。また、無効にするまで別の場所に移動しないようにしてください。

残念ながら、あなたに対して使用されたトラッカーを見つけた場合は、どこで助けを得るかを知ることが重要です。
あなたを助けることができるいくつかのネットワークや協会があります。国別の短いリストを提供します。
