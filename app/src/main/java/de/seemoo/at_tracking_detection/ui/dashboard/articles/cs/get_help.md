# Pokud jste obdrželi oznámení
### Najděte odbornou pomoc, pokud vás někdo sleduje

Pokud AirGuard na pozadí detekuje sledovač, který vás sleduje, obdržíte oznámení. Klepnutím na oznámení se dozvíte více o tomto sledovači. U podporovaných sledovačů můžete přehrát zvuk, abyste jej lokalizovali. U všech ostatních sledovačů můžete k jeho nalezení použít "Lokalizovat sledovač".

Pokud jste sledovač našli a jste si jisti, že je škodlivý, pokuste se vyjmout baterii. Také se ujistěte, že se nepřesunete na jiné místo, dokud jej nedeaktivujete.

V nešťastném případě, že jste našli sledovač, který byl použit proti vám, je důležité vědět, kde získat pomoc.
Existuje několik sítí a sdružení, které vám mohou pomoci. Poskytujeme krátký seznam podle zemí.
