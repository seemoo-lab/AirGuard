# Come funziona AirGuard?

I tracker di oggetti intelligenti utilizzano il Bluetooth per connettersi a dispositivi esterni. Questi dispositivi inviano quindi la loro posizione a Internet. Di conseguenza, il proprietario del tracker di oggetti intelligenti può vedere la posizione del tracker anche se si trova a chilometri di distanza.

Ecco perché questi tracker di oggetti intelligenti possono essere utilizzati in modo improprio per perseguitare le persone. Per proteggerti da un tale attacco, AirGuard utilizza il Bluetooth per cercare tracker nelle vicinanze. L'app utilizza la tua posizione per rilevare se un tracker si muove con te.

Se AirGuard rileva un tracker sospetto, riceverai una notifica con informazioni più dettagliate.
