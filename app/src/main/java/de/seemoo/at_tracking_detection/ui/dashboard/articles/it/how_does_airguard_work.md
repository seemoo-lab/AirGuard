# Come funziona AirGuard?

Gli smart tracker utilizzano il Bluetooth per connettersi a dispositivi esterni ed inviano poi la loro posizione a Internet. Di conseguenza, il proprietario dello smart tracker può vedere la posizione del tracker anche se si trova a chilometri di distanza.

Questo è il motivo per cui questi tracker possono essere utilizzati in modo improprio per fare stalking alle persone. Per proteggerti da un attacco di questo tipo, AirGuard utilizza il Bluetooth per rilevare i tracker nelle vicinanze. L'app utilizza la tua posizione per rilevare se un tracker si muove con te.

Se AirGuard rileva un tracker sospetto, riceverai una notifica con informazioni più dettagliate.
