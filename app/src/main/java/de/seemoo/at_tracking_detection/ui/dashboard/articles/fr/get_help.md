# Si vous avez reçu une notification
### Trouvez de l'aide professionnelle si quelqu'un vous suit

Si AirGuard détecte un tracker en arrière-plan qui vous suit, vous recevrez une notification. Vous pouvez appuyer sur la notification pour en savoir plus sur ce tracker. Pour les trackers pris en charge, vous pouvez jouer un son pour localiser le tracker. Pour tous les autres trackers, vous pouvez utiliser "Localiser le tracker" pour le trouver.

Si vous avez trouvé le tracker et que vous êtes certain qu'il est malveillant, essayez de retirer la batterie. Assurez-vous également de ne pas vous déplacer vers un autre endroit avant de l'avoir désactivé.

Dans le cas malheureux où vous auriez trouvé un tracker utilisé contre vous, il est important de savoir où obtenir de l'aide.
Il existe plusieurs réseaux et associations qui peuvent vous aider. Nous fournissons une courte liste par pays.
