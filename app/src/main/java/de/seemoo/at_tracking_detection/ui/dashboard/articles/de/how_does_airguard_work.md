# Wie funktioniert AirGuard?

Smarte Tracker verwenden Bluetooth, um sich mit fremden Geräten zu verbinden. Diese Geräte senden dann ihren Standort an das Internet. Dadurch kann der:die Besitzer:in des Trackers den Standort bestimmen, selbst wenn der Tracker viele Kilometer entfernt ist.

Das ist der Grund, warum diese Tracker auch zum Stalking von Menschen missbraucht werden können. Um dich gegen einen solchen Angriff zu schützen, nutzt AirGuard Bluetooth, um nach Trackern in der Nähe zu suchen. Dein Standort wird genutzt, um zu bestimmen, ob der Tracker sich mit dir bewegt.\n\nErkennt AirGuard einen verdächtigen Tracker, wird dir eine Benachrichtigung mit detaillierten Informationen zugestellt.
